# Nostr SDK

## Description

Nostr protocol implementation, Relay, RelayPool, high-level client library, NWC client and more.

## Documentation

Learn more about at <https://nostrdevkit.org>.

### Supported platforms

The following OS and architectures are supported:

| OS            | x86_64 | aarch64 | armv7 | i686 | riscv64 |
|---------------|--------|---------|-------|------|---------|
| Android       | ❌      | ❌       | ❌     | ❌    | ❌       |
| Linux (GLIBC) | ✅      | ✅       | ✅     | ✅    | ✅*      |
| Linux (MUSL)  | ✅      | ✅       | ✅     | ✅    | ✅*      |
| FreeBSD       | ✅*     | ✅*      | ❌     | ❌    | ❌       |
| macOS         | ✅      | ✅       | ❌     | ❌    | ❌       |
| Windows       | ✅      | ✅       | ❌     | ✅    | ❌       |

Are you interested in other platforms? Open an issue [here](https://github.com/nostrdevkit/nostr-sdk-ffi).

<small>* PyPI currently doesn't allow uploading this wheel</small>

## State

**This library is in an ALPHA state**, things that are implemented generally work but the API will change in breaking ways.
